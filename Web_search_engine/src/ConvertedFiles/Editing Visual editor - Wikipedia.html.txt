Anti-spam check.Do not fill this in!{{Multiple issues|{{Sources exist|date=February 2021}}{{Self reference|For the Wikipedia page about the VisualEditor feature, see [[Wikipedia:VisualEditor]], or to perform tests using Wikipedia's VisualEditor, go to the [[Wikipedia:Sandbox|Sandbox]] page and click "Edit page visually".}}{{Unreferenced|date=February 2019}}}}{{for|the article about the Wikimedia editor project|VisualEditor}}A '''visual editor''', or '''full-screen editor''' is [[computer software]] for editing [[ASCII|text files]] using a [[text user interface|textual]] or [[graphical user interface|graphical]] user interface which displays the content (text) in an easy to look at and good view; that is, it displays a portion of the opened file and updates it in [[wikt:real-time|real time]].  By definition, all visual editors require a re-fresh-able display device and all editors with a GUI are visual.
